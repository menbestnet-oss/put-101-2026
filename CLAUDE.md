## Как обновить журналы
1. Экспортировать чат из Telegram (HTML) → сохранить по тому же пути
2. Запустить `ОБНОВИТЬ_ЖУРНАЛЫ.bat` или `python parse_journals.py`
3. Скрипт сам пушит на GitHub → через 1-2 мин сайт обновляется

## Дизайн журналов
- Тёмная тема (#0d0d1a фон)
- Аккордеон-карточки на каждый день
- Блоки: синий = утро, зелёный = рефлексия, жёлтый = вывод, фиолетовый = прочее
- Бейджи факта: красный (0), оранжевый (мало), зелёный (хорошо), зелёный с рамкой (план закрыт)
