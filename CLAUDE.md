# Путь 101% 2026 — Журналы команды

## Что это
Личные HTML-журналы для 9 менеджеров по продажам. Данные берутся из темы
«Путь 101 % 2026» Telegram-группы «Отдел продаж📈🔥💰».
Сайт опубликован на GitHub Pages: https://menbestnet-oss.github.io/put-101-2026/

## Как работает автомат
```
Планировщик Windows (PUT101_Journals_Update, каждые 2 ч с 08:00 до 22:00)
        │
        ▼
auto_update.py
  1. fetch_telegram.py — Telethon забирает новые сообщения темы → messages_cache.json
  2. parse_journals.py — пересборка журналов (только если есть новые сообщения)
  3. git commit + push — только при реальных изменениях (пустых коммитов нет)
```
Лог: `auto_update.log` (в этой папке).


## Планировщик
- Задача: `PUT101_Journals_Update`, ежедневно 08:00, повтор каждые 2 ч до 22:00
- Запуск: pythonw.exe (без окна), только при залогиненном пользователе
- Проверить: `schtasks /query /tn PUT101_Journals_Update`
- Запустить сейчас: `schtasks /run /tn PUT101_Journals_Update`
- Удалить: `schtasks /delete /tn PUT101_Journals_Update /f`
